Open Console

Visual Studio Code is an IDE. It is an offline code editor which means that you can work in this code editor without internet.It is used by many people in the world.

The console screen is very helpful when you are coding.It allows you to test your website.Console screen also allows you to correct your mistakes. To open the console screen, right click on the website and click on inspect.After that the inspect screen occurs and then finally click console.To print anything to the console screen go to the IDE and type "console.log()" and in the bracket type the variable or the sentence you want to print.
